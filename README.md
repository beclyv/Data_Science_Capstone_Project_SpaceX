# SpaceX Falcon 9 Launch Records Dashboard and Predictive Analysis

## Project Overview
This capstone project analyzes SpaceX Falcon 9 launch records to explore the factors influencing launch success.  
It combines **data collection, wrangling, exploratory analysis, interactive dashboards, mapping, and predictive modeling** and shows the code written in the labs for the IBM Data Science Certificate.

Repository Structure is as follows:

- `data/` → datasets (CSV, JSON)
- `notebooks/` → Jupyter notebooks for each lab
- `dashboard/` → Dash app files
- `results/` → exported charts/screenshots
- `ppt/` → capstone presentation
- `requirements.txt` → Python dependencies
